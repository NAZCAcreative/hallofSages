# Game assets

Drop PNG files here. Missing files fall back automatically (rectangles / grid),
so the game runs with or without them.

| 파일명 (정확히 이 이름) | 용도 | 권장 크기 |
|------------------------|------|-----------|
| background.png         | 게임 배경 (전체 화면) | 960×600 (가로:세로 8:5) |
| jesus.png              | 예수님 캐릭터 | 정사각형 권장 (예: 256×256), 투명 배경 |
| buddha.png             | 부처님 캐릭터 | 정사각형 권장, 투명 배경 |
| confucius.png          | 공자님 캐릭터 | 정사각형 권장, 투명 배경 |
| player.png             | 방문자 캐릭터 (선택) | 정사각형 권장, 투명 배경 |

- 캐릭터는 화면에서 최대 72px로(비율 유지) 표시됩니다. 원본은 더 커도 됩니다.
- 배경은 960×600으로 늘려 채웁니다. 다른 비율이면 늘어나 보일 수 있어요.
